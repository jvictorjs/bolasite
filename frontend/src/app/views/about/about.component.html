Aplicação web SPA desenvolvida em ANGULAR, um framework que roda sob NODE.JS, usando TYPESCRIPT, HTML e CSS. Adicionalmente, usei FONT AWESOME para alguns ícones. Hospedado em um servidor Linux UBUNTU na nuvem da DIGITALOCEAN. Controle de versão feito com GIT e GITHUB. Backend rodando sob JSON SERVER e também GOOGLE APPS SCRIPTS. Gerenciamento de pacotes com NPM. Proxy reverso NGINX, certificação SSL com CERTSSL e gerenciamento de processos Linux com PM2.
